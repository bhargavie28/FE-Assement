# React Assesment

### How to run


1. Clone this repository

```bash
git clone https://github.com/bhargavie28/fe-assement.git
```

2. Install packages

```bash
npm install
```

3. Run the server

```bash
npm run dev
```

### How to run tests

```bash
npm test
```

#### To check Coverage, Run

```bash
npm test:coverage
```
